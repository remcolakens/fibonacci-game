# Fibonacci Game

Before you get started make sure you have the right versions of node & yarn installed on your computer.

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```
